# Practical 5  Network Defense with Open-Source Tools

## Objective
Detect and block malicious network activity.

## Tools Used
- Suricata
- Elastic SIEM
- CrowdSec

## Workflow
1. Configure Suricata IDS rules.
2. Detect malicious traffic.
3. Map alerts to MITRE ATT&CK.
4. Block attacker IP using CrowdSec.
5. Validate containment.

## Outcome
Malicious IP was detected and blocked.
