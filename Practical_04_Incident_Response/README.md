# Practical 4  Incident Response Simulation

## Objective
Simulate phishing attack and collect forensic artifacts.

## Tools Used
- MITRE Caldera
- Velociraptor

## Workflow
1. Deploy phishing simulation using Caldera.
2. Trigger attack on Windows VM.
3. Collect artifacts using Velociraptor.
4. Export results to CSV.
5. Analyze indicators of compromise.

## Outcome
Attack artifacts were collected and analyzed successfully.
