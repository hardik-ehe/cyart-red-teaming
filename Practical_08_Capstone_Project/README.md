# Practical 8  Capstone: Full Incident Response Cycle

## Objective
Simulate, detect, contain, and report a real attack.

## Tools Used
- Metasploit
- Wazuh
- CrowdSec

## Workflow
1. Exploit Metasploitable2 using Metasploit.
2. Detect attack via Wazuh alerts.
3. Block attacker IP using CrowdSec.
4. Verify containment.
5. Prepare incident report.

## Outcome
End-to-end incident response lifecycle completed.
