# CYART Red Teaming – Week 2 Practicals

This repository contains **hands-on red teaming, incident response, and defensive security practicals** performed as part of CYART Week 2.  

Each practical includes:
- A detailed PDF report
- Step-by-step workflow documentation (`README.md` in each folder)
- Screenshots as evidence of execution
- Supporting files where applicable

---

3. Vulnerability Management

5. Network Defense  
6. Risk Assessment  
7. Incident Response Reports  
8. Capstone Project

## 🛠 Tools & Technologies Used

- **Threat Hunting & SIEM:** Elastic Security, Security Onion, Sigma Rules  
- **Malware Analysis:** REMnux, Hybrid Analysis  
- **Vulnerability Management:** OpenVAS, DefectDojo  
- **Incident Response:** MITRE Caldera, Velociraptor  
- **Network Defense:** Suricata, Elastic SIEM, CrowdSec  
- **Red Team & Detection:** Metasploit, Wazuh  
- **Documentation & Reporting:** Google Docs, Google Sheets, PDF exports

---

## 🎯 Learning Outcomes

- Conduct threat hunting using open-source tools  
- Perform static and dynamic malware analysis  
- Scan for vulnerabilities and prioritize remediation  
- Simulate phishing and incident response workflows  
- Configure network defense rules and map alerts to MITRE ATT&CK  
- Calculate risk assessment metrics (ALE, Risk Matrix)  
- Produce detailed incident response reports and capstone documentation  
- Execute a full red team simulation, detect, contain, and report attacks

---

## 📌 Notes

- All activities were performed in **controlled lab environments** using intentionally vulnerable systems and safe simulations.  
- Screenshots and PDFs serve as **evidence of execution**.  
- This repository is for **educational purposes only**.

## 📝 How to Navigate

1. Open each `Practical_XX` folder.  
2. Read the `README.md` in that folder for **workflow steps**.  
3. Open PDF for **detailed report and screenshots**.  
4. For practical 6, open the Excel file for **risk assessment calculations**.  
5. Practical 8 (capstone) contains the **full incident response simulation workflow**.
